# Reception Management Dashboard

## Project Description
This web application was developed as part of Noroff Backend Development Year 1 Semester Project.

The system allows reception staff to:
- Manage staff availability
- Track delivery drivers
- Receive late notifications
- View real-time date and time

## Installation Instructions

1. Clone repository:
git clone <repo-link>

2. Open project in Visual Studio Code.

3. Install **Live Server** extension.

4. Right-click `index.html`.

5. Select **Open with Live Server**.
